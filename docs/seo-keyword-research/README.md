# サクッと印紙税 — SEOキーワード調査

Phase 1 title/meta 最適化 + Phase 2 FAQ拡充のためのキーワード調査ログ置き場。

## 保存してほしいファイル

| ファイル名 | 内容 | 取得元 |
|-----------|------|--------|
| `rakkokeyword_relatedKeywords_印紙税_YYYY-MM-DD_*.csv` | 「印紙税」のラッコサジェスト（月間検索数・SEO難易度付き） | ラッコキーワード有料プラン |

## ラッコキーワード操作手順

1. https://related-keywords.com/ にアクセス・ログイン
2. 検索窓に `印紙税` と入力 → 「無料で調査」
3. 「月間検索数/SEO難易度（最新）を取得」ボタンをクリック → 数値が列に追加される
4. マッチタイプ: 部分一致（デフォルト）、**フィルタは解除したまま**（5,000件全体を取得）
5. 画面右上「データ出力」→ **CSV** → ダウンロード → このフォルダに移動

## 想定 sweet spot 範囲

印紙税の場合、以下のようなロングテール語が見つかる想定：
- 「印紙税 契約書」「印紙税 領収書」（基本論点）
- 「印紙税 5万円」「印紙税 10万円」「印紙税 100万円」（金額別、ツールで答えられる）
- 「印紙税 第1号文書」「第7号文書」（号別、ツールの強み）
- 「不動産 売買 印紙税」「消費貸借 印紙税」（契約種別）
- 「印紙税 軽減措置」（現行制度）
- 「電子契約 印紙税」（非課税ニーズ）

特に **金額別** と **号別** は 2,400-5,000件の中でお宝が多い想定（経費判定の「メガネ 経費」と同じパターン）。
