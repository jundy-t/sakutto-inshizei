## 1. 一次典拠

### 1.1 印紙税法 別表第一 末尾「通則」

- 正本: [e-Gov 印紙税法 342AC0000000023](https://laws.e-gov.go.jp/law/342AC0000000023)
- ローカル保存: `_shared/data/sources/nta-inshizei/_raw/egov/inshizei-ho.xml`
- 通則3 が「1通の文書が2以上の号に該当する場合の所属」を定める中核条文

通則3 の構造（条文順）:
- **(イ)** 第1号 または 第2号 vs 第3〜17号 → 原則 第1号/第2号
  - 例外1: 第1号/2号に金額記載なし + 第7号該当 → 第7号
  - 例外2: 第1号/2号に金額記載なし + 第17号売上代金100万円超 → 第17号
- **(ロ)** 第1号 vs 第2号 → 原則 第1号（金額区分可能で 2号金額 > 1号金額なら 2号）
- **(ハ)** 第3号〜第17号同士 → 号数の小さい方（17号売上代金100万円超なら 17号）
- **(ニ)** 第1〜17号（証書）vs 第18〜20号（通帳等）→ 通帳等が優先
- **(ホ)** 第19号・第20号 + 高額証書（1号10万超 / 2号100万超 / 17号100万超）→ 証書側優先

### 1.2 国税庁 印紙税法基本通達 第11〜14条

- 親ページ: [国税庁 印紙税法基本通達](https://www.nta.go.jp/law/tsutatsu/kihon/inshi/mokuji.htm)
- ローカル保存: `_shared/data/sources/nta-inshizei/_raw/html/nta-kihon-tsutatsu-key-articles.txt`
- 第11条: 通則3 の所属決定の詳細運用
- 第12条〜第14条: 金額記載なしの取扱い、金額区分の可否判定、混合文書の境界事例

### 1.3 国税庁 タックスアンサー

- No.7100（印紙税の手引）: https://www.nta.go.jp/taxes/shiraberu/taxanswer/inshi/7100.htm
- No.7140/7141（印紙税額の一覧表）: 通則3 による号所属の例示
- ローカル保存: `_shared/data/sources/nta-inshizei/_raw/html/nta-7140-7141-tax-table.txt`

### 2.1 金額区分（通則3ロ例外）の実装

通則3ロには「金額区分ができて2号金額が1号金額を超える場合は2号」という例外がある。

**「金額区分可能」の定義（通則4）:**
> 2以上の号に該当し金額区分可能 → 所属する号の金額のみが記載金額となる
> 区分不能 → 全額（他号の金額として明示部分は除く）

出典: `_shared/data/sources/nta-inshizei/_raw/html/egov-inshizei-betsu1-tsusoku.txt` line 87-88

**実装方式:**
- 追加検討アコーディオンで1号系 + 2号 の両方が該当した場合のみ、任意のサブ質問を表示
- ユーザーが「金額が別々に記載されている」を選び、各号分の金額を入力した場合に `amountByClass` として `applyRule3()` に渡す
- `applyRule3()` 内の `applyRule3Ro()` 関数で、2号金額 > 1号金額 なら第2号に所属（例外発動）
- 金額を入力しない場合は原則どおり第1号優先（保守的判断）

### 2.2 実装上の保守的判断（原典からの乖離がある点）

以下は**保守的に「当初号を維持 / 原則どおり」に倒す**実装にしている：

- 通則3イの「17号売上代金100万円超が1号/2号金額を超える」例外 → 金額記載ありの場合は実装では判定せず、当初号維持。理由: 契約書+領収書の複合は実務頻度が低く、金額区分UIの複雑化に見合わない
- 通則3ホ（高額証書の逆転） → 本エンジンは対象外（通帳等と併存するケース自体が本ツールのカバー範囲外）
- 軽減措置の再適用 → 最終号が変更された場合、軽減措置は不適用とする（保守的）。理由: 軽減措置は号+取引性質の組み合わせで決まり、ハイブリッド文書の場合の適否は専門家判断が必要

UI 側では結果欄に「判定が微妙な場合は税理士・行政書士・税務署への相談を推奨」の注記を出しており、厳密判定は専門家導線に委ねる設計。

`_shared/data/sources/nta-inshizei/data.ts` の `HYBRID_OPTIONS` で定義している4つの対象号 × 各2〜3候補の根拠：

| 対象号 | 併存候補 | 根拠 |

## 4. ツール対象外（意図的スコープ外）

以下は今回の実装では扱わないことを明示：

- **基本通達レベルの境界判定**（例: 請負 vs 売買の実質判定、請負 vs 委任の実質判定）→ 専門家相談へ
- **通則3ホ**（第19/20号 + 高額証書）→ 対象ユーザー（個人事業主）の利用頻度が極めて低い
- **通則3イ例外2の金額比較** → 1号/2号 vs 17号の金額比較は実務頻度が低く、注記で誘導

これらは UI 上「判定が微妙な場合は税理士・行政書士・税務署への相談を推奨」の注記で誘導する。
